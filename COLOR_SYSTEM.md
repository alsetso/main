# Alset Color System

## Standardized Color Palette

This document defines the standardized white, black, and gray color system used throughout the Alset website.

### Background Colors

- **`bg-white`** - Pure white (#FFFFFF)
  - Use for: Main content areas, cards, primary backgrounds
  - Provides maximum contrast and clean appearance

- **`bg-gray-50`** - Very light gray (#F9FAFB)
  - Use for: Subtle section backgrounds, alternating sections
  - Creates gentle visual separation without harsh contrast

- **`bg-gray-100`** - Light gray (#F3F4F6)
  - Use for: Badges, subtle UI elements, card backgrounds, hover states
  - Provides subtle emphasis without being too prominent

- **`bg-gray-900`** - Dark gray (#111827)
  - Use for: Dark sections, call-to-action backgrounds
  - Alternative to black for softer dark appearance

- **`bg-black`** - Pure black (#000000)
  - Use for: Footer, high-contrast sections, strong emphasis
  - Maximum contrast and bold statement

### Text Colors

- **`text-black`** - Pure black (#000000)
  - Use for: Primary headings, maximum emphasis
  - Best for: Large display text, hero headlines

- **`text-gray-900`** - Near-black (#111827)
  - Use for: Main body text, headings, primary content
  - Standard for: Most text on white/light backgrounds
  - Provides excellent readability

- **`text-gray-700`** - Dark gray (#374151)
  - Use for: Secondary body text, supporting content
  - Good for: Paragraphs, descriptions, less critical information

- **`text-gray-600`** - Medium gray (#4B5563)
  - Use for: Tertiary text, captions, metadata
  - Use when: Text is less important but still needs to be readable

- **`text-gray-500`** - Muted gray (#6B7280)
  - Use for: Muted text, copyright, timestamps
  - Use when: Text is informational but not critical

- **`text-gray-400`** - Light gray (#9CA3AF)
  - Use for: Subtle text on dark backgrounds
  - Good for: Footer descriptions, secondary info on black/gray-900

- **`text-white`** - Pure white (#FFFFFF)
  - Use for: All text on black or gray-900 backgrounds
  - Essential for: Footer, dark CTAs, inverted sections

### Border Colors

- **`border-gray-200`** - Very light gray (#E5E7EB)
  - Use for: Subtle borders, card borders, dividers
  - Standard for: Most UI elements on white backgrounds

- **`border-gray-300`** - Light gray (#D1D5DB)
  - Use for: More visible borders, input fields
  - Use when: Border needs more definition

- **`border-gray-800`** - Dark gray (#1F2937)
  - Use for: Borders on dark backgrounds (black, gray-900)
  - Provides subtle separation on dark sections

### Usage Guidelines

1. **Contrast Requirements:**
   - Text on white: Use `text-gray-900` or `text-black` for body text
   - Text on gray-50: Use `text-gray-900` or `text-gray-700`
   - Text on gray-100: Use `text-gray-900` or `text-gray-700`
   - Text on black: Always use `text-white`
   - Text on gray-900: Use `text-white` or `text-gray-400`

2. **Hierarchy:**
   - Primary content: `text-gray-900` or `text-black`
   - Secondary content: `text-gray-700`
   - Tertiary content: `text-gray-600`
   - Muted content: `text-gray-500`

3. **Consistency:**
   - Always use the same gray shades for similar purposes
   - Maintain consistent contrast ratios for accessibility
   - Use black and white sparingly for maximum impact
